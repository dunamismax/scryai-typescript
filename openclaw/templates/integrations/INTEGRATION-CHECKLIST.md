# Integration Checklist (Per Repo)

Copy into each repo `BUILD.md` or keep as a linked checklist.

- [ ] Business value defined (what this integration unlocks)
- [ ] Required credentials/scopes documented
- [ ] Data classification and exposure risk noted
- [ ] Auto-act vs ask-first behavior decided
- [ ] Rollback/disconnect procedure documented
- [ ] Monitoring/alerts defined
- [ ] BUILD.md updated with current integration state
